# Fake News Detection

A simple Machine Learning project that detects whether a news article is Fake or Real using Natural Language Processing (NLP) techniques.

## Features

- Text preprocessing using TF-IDF Vectorization
- Fake and Real news classification
- Logistic Regression model
- Model evaluation using Accuracy Score
- Custom news prediction

## Tech Stack

- Python
- Pandas
- NumPy
- Scikit-learn
- Jupyter Notebook

## Dataset

The project uses the Fake and Real News Dataset containing:

- Fake.csv
- True.csv

Dataset Source:
https://www.kaggle.com/datasets/clmentbisaillon/fake-and-real-news-dataset/data


## Machine Learning Pipeline

1. Load Dataset
2. Data Preprocessing
3. TF-IDF Vectorization
4. Train-Test Split
5. Logistic Regression Training
6. Model Evaluation
7. News Prediction

## Results

The model achieves high accuracy (98.27%) on the test dataset using TF-IDF and Logistic Regression.
